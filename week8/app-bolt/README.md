# Bolt Version – Full Stack Task Manager

This version of the Task Manager application was generated using **Bolt AI**.  
It demonstrates a modern AI-assisted workflow for quickly building a functional web application.

The app provides a simple interface for managing tasks, including creating, viewing, updating, and deleting tasks.
